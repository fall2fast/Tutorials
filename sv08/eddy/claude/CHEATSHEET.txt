┌───────────────────────────────────────────────────────────────────┐
│                          THE FIX                                  │
└───────────────────────────────────────────────────────────────────┘
Change ONE LINE in printer.cfg:
  FROM: i2c_bus: i2c1
  TO:   i2c_bus: i2c1_PB6_PB7

┌───────────────────────────────────────────────────────────────────┐
│                    INSTALLATION SEQUENCE                          │
└───────────────────────────────────────────────────────────────────┘
1. Upload fixed printer.cfg
2. RESTART
3. QUERY_PROBE (should return "probe: open")
4. G28
5. Z_OFFSET_CALIBRATION (paper test)
6. G28
7. QUAD_GANTRY_LEVEL
8. BED_MESH_CALIBRATE (wait 5 min)
9. SAVE_CONFIG
10. Test print!

┌───────────────────────────────────────────────────────────────────┐
│                     ESSENTIAL COMMANDS                            │
└───────────────────────────────────────────────────────────────────┘
Check probe:          QUERY_PROBE
Home:                 G28
Calibrate offset:     Z_OFFSET_CALIBRATION
Level gantry:         QUAD_GANTRY_LEVEL
Mesh bed:             BED_MESH_CALIBRATE
Save config:          SAVE_CONFIG

Adjust Z live:
  Up:   SET_GCODE_OFFSET Z=+0.05
  Down: SET_GCODE_OFFSET Z=-0.05

┌───────────────────────────────────────────────────────────────────┐
│                      PRO TIPS                                     │
└───────────────────────────────────────────────────────────────────┘
• Always QGL before meshing
• Heat soak bed 10 min before mesh
• Clean PEI sheet before mesh
• Don't interrupt 41×41 mesh (5 min)
• Save mesh as default after success
• Re-mesh if temp changes >10°C
• Trust the Eddy - it's more accurate!
